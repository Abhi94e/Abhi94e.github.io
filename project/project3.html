Jeux WEB

Introduction

Ce projet consiste en un jeu web interactif développé à l'aide de technologies de base du web : HTML, CSS, et JavaScript. Le jeu est simple : le joueur doit cliquer sur des cibles (représentées par une image de cafard) qui apparaissent aléatoirement sur l'écran, tout en essayant d'obtenir le score le plus élevé possible avant la fin du temps imparti. Ce document présente les trois fichiers principaux du projet : index.html, style.css, et script.js, en détaillant leur structure et leur rôle dans le jeu.

Fichier JavaScript

Le fichier script.js contient la logique de jeu, gérant les événements et les interactions de l'utilisateur.

            
    // Sélection des éléments HTML nécessaires pour le jeu
    let container = document.querySelector('.container')
    let btn = document.querySelector('.start_btn')
    let scoreContainer = document.querySelector('.score')
    let timeContainer = document.querySelector('.time')

    // Fonction pour démarrer le jeu
    btn.onclick = function() {
        let score = 0 // Initialisation du score
        let time = 20 // Temps de jeu initial en secondes
        container.innerHTML = '' // Réinitialisation du conteneur

        // Intervalle pour afficher les cibles à chaque seconde
        let interval = setInterval(function showTarget() {

            // Création et positionnement de la cible
            let target = document.createElement('img')
            target.id = 'target'
            target.src = 'target.png'
            container.appendChild(target)
            target.style.top = Math.random() * (400 - target.offsetHeight) + 'px'
            target.style.left = Math.random() * (500 - target.offsetWidth) + 'px'

            // Suppression de la cible après 2 secondes
            setTimeout(function() {
                target.remove()
            }, 2000)

            // Incrémentation du score au clic sur la cible
            target.onclick = function() {
                score += 1
                target.style.display = 'none'
            }

            // Mise à jour du temps restant
            time -= 1

            // Mise à jour des affichages du score et du temps
            scoreContainer.innerHTML = `Score : ${score}`
            timeContainer.innerHTML = `Temps : ${time}`

            // Vérification si le temps est écoulé
            if (time == 0) {
                clearInterval(interval); // Arrêt de l'intervalle
                container.innerHTML = 'le jeu est terminé'; // Message de fin de jeu
            }
        }, 1000) // Intervalle d'une seconde
    }

Code PHP

            
            <?php
    $nmb = rand(0, 100);

    if (isset($_GET['nmb'])) {
        $user_nmb = (int) $_GET['nmb'];

        if ($user_nmb < $nmb) {
            echo '<b>Perdu!</b> Votre nombre est trop petit. le nombre été : ' . $nmb;
        } elseif ($user_nmb > $nmb) {
            echo '<b>Perdu!</b> Votre nombre est trop grand. Vous avez essayé le nombre : ' . $user_nmb;
        } elseif ($user_nmb === $nmb) {
            echo '<b>C'est exact</b>, le compte est bon !  le nombre été : ' . $nmb;
        }
    } else {
        echo "Essayez !";
    }
    ?>
